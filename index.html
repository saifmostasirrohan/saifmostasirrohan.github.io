PROJECT_01  ·  AUTONOMOUS AGENT SYSTEM
PROJECT AEGIS
DEPLOYED

Project Aegis is a staged autonomous research assistant that evolved from a domain chatbot into a production-style LangGraph workflow. It searches ArXiv and local research stores, retrieves evidence, compares papers, pauses for human approval, and writes structured literature-review drafts. The deployment uses a single Hugging Face Spaces container with Streamlit on the public port and FastAPI running internally.

PHASE BREAKDOWN
PH-01 ✓
Research Query Chatbot
Groq streaming, Pydantic structured outputs, SQLite multi-turn memory, and context-window trimming.
PH-02 ✓
Hybrid RAG Pipeline
Vector search + BM25 fused through Reciprocal Rank Fusion, cross-encoder reranking, and FAISS indexing.
PH-03 ✓
Fine-Tuned Local Medical Model
Quantized 3B GGUF model through llama.cpp with ChatML formatting and structured JSON diagnostic reports.
PH-04 ✓
Autonomous Research Assistant
LangGraph state machine: search → read → compare → human interrupt → write. FastMCP tools, ArXiv integration, ChromaDB, and HF Spaces deployment.
PRODUCTION UPGRADES
OPS
Unified Docker Deployment
One container starts both Streamlit and FastAPI, monitors both processes, and gracefully shuts down on failure.
SAFE
Graceful Startup + Keyless Mode
Mock-key protection prevents startup crashes, while optional API keys enforce strict bearer-token security when configured.
OBS
Evaluation and Observability
LLM-as-judge quality gate, structured logs, Prometheus metrics, Grafana dashboard, and LangSmith traces.

PROJECT_02  ·  MULTI-AGENT FINANCIAL SYSTEM
FINSIGHT
IN PROGRESS

FinSight is an autonomous financial audit and fraud-detection network. Users upload PDFs or CSVs, ask audit questions in natural language, and the system routes work through specialized agents for retrieval, analysis, action logging, fraud scoring, compliance checking, and response generation.

PHASE TRACKER
PH-01 ✓
Foundation
FastAPI backend, LangGraph agent shell, Groq integration, Pydantic models, and basic chat endpoint.
PH-02 ▶
Document Intelligence & RAG
PDF/CSV upload, chunking, sentence-transformers embeddings, Supabase pgvector storage, and semantic search.
PH-03
The Agent Network
Five specialized agents: Router → Retrieval → Analysis → Action → Response.
PH-04
Financial Intelligence
Duplicate detection, anomaly scoring, compliance checking, fraud risk scoring, human approval, and audit trail logging.
PH-05
Memory & Session Management
Cross-session memory, LangGraph SQLite checkpointer, previous audit recall, and user preference store.
PH-06
Monitoring & Quality Gates
Structured logs, Prometheus, Grafana, pytest, GitHub Actions CI/CD, and LLM-as-judge evaluation.
PH-07
Deployment — Live Public URL
Dockerized FastAPI backend, Streamlit frontend, HF Spaces, Render backend, Supabase cloud, and shareable public link.
